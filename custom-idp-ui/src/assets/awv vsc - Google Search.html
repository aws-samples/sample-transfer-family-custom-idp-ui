    What are the different types of AWV?
    There are three types of visits:
    • Initial Preventive Physical Exam (IPPE)
    • Initial Annual Wellness Visit (AWV)
    • Subsequent AWVs.

    What are possible AWV codes?
    AWV Coding. The CPT codes for Annual Wellness Visits are G0438 for the initial visit and G0439 for subsequent visits. These codes are used to bill Medicare for comprehensive wellness assessments and personalized prevention plans. It is important to remember that code G0438 is for the first AWV only.

    What codes can be billed with G0439?
    HCPCS G0439 is used to code all subsequent Medicare annual wellness visits that occur after the initial AWV (G0438). So, if used correctly, G0439 would not be used until G0402 was used to code the IPPE and G0438 was used to code the initial AWV.Mar 1, 2023
